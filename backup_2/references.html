Website

New members: If you're interested in web development, complete the following steps.

  1. Sign up for a GitHub account
  2. Setup Git on your machine.
  3. Read the Learn Enough Git tutorial and understand the basics. You may also want to read their Command Line tutorial and Text Editor tutorial if you're unfamiliar with those tools.
  4. Email admin@ubcchemecar.com for further instructions once you feel comfortable with the basics.
